## Фабрика (Factory)

Фабричный паттерн предоставляет интерфейс для создания объектов в суперклассе, но позволяет подклассам изменять тип создаваемых объектов.

## Абстрактная фабрика (Abstract Factory)

Абстрактная фабрика позволяет создавать семейства связанных или взаимозависимых объектов без указания их конкретных классов.


## Строитель (Builder)

Строительный паттерн позволяет создавать сложные объекты пошагово, позволяя использовать один и тот же процесс конструирования для различных представлений объектов.
